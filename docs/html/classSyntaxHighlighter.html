Detailed Description

Multi-language syntax highlighter for the text editor.

SyntaxHighlighter extends QSyntaxHighlighter to provide syntax highlighting for multiple programming languages:

  • C/C++ with keywords, preprocessor directives, and comments
  • Python with indentation-aware highlighting
  • JavaScript with modern ES6+ features
  • JSON with proper structure highlighting
  • XML with tag and attribute highlighting
  • Plain text (no highlighting)

The highlighter automatically detects language based on file extension and applies appropriate color schemes for different syntax elements.
