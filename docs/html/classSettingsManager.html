Detailed Description

Manages persistent application settings and session data.

SettingsManager provides comprehensive settings persistence using QSettings. It handles:

  • Window geometry and state preservation
  • Recent files list maintenance
  • Editor preferences (font, theme, behavior)
  • Session management with complete tab restoration
  • Auto-save content for crash recovery

The class uses the application's organization and name for settings storage, ensuring settings persist across application runs and system reboots.
